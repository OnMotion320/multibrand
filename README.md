# Release Form App

A Vite + React release-form app built for iPad and web use.

## Included features
- Adult model, minor model, and property release types
- Company-based release language
- Parent / guardian signer flow for minors
- Photo attachment / visual reference
- Signature pad
- Terms acceptance checkbox
- PDF generation with PNG logo support
- Local history in browser storage

## Run locally
```bash
npm install
npm run dev
```

## Vercel settings
- Framework Preset: Vite
- Build Command: npm run build
- Output Directory: dist
- Install Command: npm install

The app uses:

## Notes
- Saved history is local to that browser/device.
- This is a strong practical workflow for commercial/stock preparation, but for final legal review you should still have an attorney review the release wording for your business and jurisdictions.
